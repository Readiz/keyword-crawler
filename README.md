## installation
using npm
```bash
npm i
```

using yarn
```bash
yarn
```

## develop
### run task runner(gulp)
1. eslint  
2. babel

using npm
```bash
npm run dev
```

using yarn
```bash
yarn dev
```

### run crawler(phantomjs-prebuilt)
using npm
```bash
npm start
```

using yarn
```bash
yarn start
```
